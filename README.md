# Hands on I - Exploratory Data Analysis of COVID patients Metadata

This repository contains the analysis for **Hands on I (DMI course)**.  
The objective of the project is to determine tendencies and significant differences between patient groups, as well as possible biases and findings in the behaviour of the Sars-Cov-2 disease.

The analysis is based on the dataset from the publication:  
**Proteomic and Metabolomic Characterization of COVID-19 Patient Sera**  
Shen et al., Cell (2020)

---

# Report

The full analysis report is available here:  
➡ **[Hands_on_I.html](Hands_on_I.html)**

The HTML report contains:
- Data preprocessing and quality control. (Exercise 1)
    - Reproduction of TableS1 (Exercise 1.1)
    - Exploratory Data Analysis of the Metadata (Exercise 1.2)
    - Studying Data distribution.
    - Correlation between variables.
- Reproduction of figure S1 (Exercise 2)
    - Data setup and plotting (Exercise 2.1)
    - Outliers handling and depiction (Exercise 2.2)
- Biomarkers heatmap (Exercise 3)

The report is fully self-contained and includes all figures.
